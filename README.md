# Grandpa's New Ears

中文工作名：爷爷的新耳朵

Grandpa's New Ears 是一款面向听障人士、听力下降老人及其家人的实时字幕应用。核心体验很简单：用户打开自己的手机或平板，对方开始说话，屏幕上立即出现清晰、足够大的文字。

## 核心方向

- 面对面谈话实时转文字。
- 大字号、高对比度、低干扰的字幕阅读体验。
- 可选翻译能力，支持跨语言沟通。
- 默认不保存对话，只有用户明确选择时才保存。
- 先用移动端网页或 PWA 快速验证，再根据反馈决定是否做原生 App。
